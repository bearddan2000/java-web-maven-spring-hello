# java-web-maven-spring-hello

## Description
A POC for springboot web app.

## Tech stack
- java
- maven
  - springboot

## Docker stack
- openjdk:11-jdk

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
